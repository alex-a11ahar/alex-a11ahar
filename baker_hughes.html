Baker Hughes Data Downsampling

GitHub-Repository

This project explores advanced data downsampling techniques to efficiently reduce large datasets while preserving key statistical properties. The dataset, containing frequency (FREQ) and power (POWER) measurements, is analyzed using exploratory data analysis (EDA) techniques like scatter plots, hexbin plots, and Kernel Density Estimation (KDE) to identify density patterns. Two distinct downsampling methods are implemented: uniform grid-based downsampling, which ensures even coverage across the dataset, and high-density clustering-based downsampling, which prioritizes retaining critical high-density regions using K-Means and Gaussian Mixture Model (GMM) clustering. The project also applies Pearson correlation and nearest neighbor analysis to evaluate the effectiveness of each approach.

Results show that uniform downsampling maintains even coverage but may discard high-density regions, while high-density clustering-based downsampling better retains essential patterns at the cost of increased computation. The Elbow Method and Bayesian Information Criterion (BIC) determine the optimal number of clusters for K-Means and GMM, respectively. Heatmap comparisons and correlation analysis confirm that the clustering-based approach achieves a 99.99% similarity in cluster density distributions with the original dataset. This project demonstrates the effectiveness of intelligent data reduction strategies, with potential applications in signal processing, anomaly detection, and industrial data science.
