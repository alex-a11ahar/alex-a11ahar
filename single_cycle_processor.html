Single Cycle Processor

GitHub-Repository

The Single-Cycle Processor project involved designing and implementing a custom processor architecture in Verilog, integrating key components such as an Arithmetic Logic Unit (ALU), control unit, register file, and memory modules. The processor was designed to execute instructions in a single cycle, ensuring efficient operation and streamlined data processing.

To ensure correctness and reliability, the processor was rigorously verified, validated, and debugged using VCS waveforms and testbenches, allowing for in-depth analysis of signal behavior and functional accuracy. Additionally, a watchdog timer was implemented for cycle management and error handling in simulation, enhancing fault detection and system stability. This project demonstrates proficiency in digital design, hardware verification, and processor architecture, showcasing a strong foundation in computer engineering principles.
