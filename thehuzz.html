TheHuzz

GitHub-Repository

The TheHuzz Optimization Project focused on enhancing hardware fuzzing efficiency through dynamic mutation and feedback-driven fuzzing. By refining the mutation engine, the project improved coverage while reducing runtime, enabling more effective vulnerability detection in hardware verification. A key contribution was the development of an effectiveness-checking algorithm, which optimized mutation strategies by identifying and eliminating redundant mutations, leading to more targeted and efficient test case generation.

To validate these improvements, the optimized mutation engine was benchmarked against existing verification methods at the Register Transfer Level (RTL) on open-source processors within a Linux environment. Experimental results demonstrated up to a 7.95% increase in coverage, showcasing the impact of adaptive fuzzing techniques in accelerating and improving hardware security validation.
